## Version Number Format

```none
YYYY.RELEASE.FIX
```

- YYYY represents the year
- RELEASE indicates the number of releases made in that year
- FIX starts at 0 and increments when bug fixes are applied
- Example version numbers:
  - 2021.1.0
  - 2021.3.1

### Canary Versions

```none
YYYY.RELEASE.FIX-canary.CANARY-RELEASE
```

- We may periodically release versions in development as "Canary" versions prior to official release
- CANARY-RELEASE starts at 0 and increments sequentially (1, 2, 3, ...) until an official release is made

- Example version numbers:
  - 2021.1.0-canary.0
  - 2021.3.1-canary.3

## Questions

**Please read the documentation and test the software first before asking questions**
Submit your questions to the appropriate channels on Discord.
If you are unsure which Discord channel to post in,
please check the `#help` channel to determine where to post.

Shiguredo members will prioritize responding to questions from users who **have the contribution label**.

For inquiries about programming languages or framework usage, please refer to [Stack Overflow](https://stackoverflow.com/) or other relevant resources.

### If you are experiencing an issue and need assistance

Please specify what information would help resolve your problem. Be sure to include detailed steps to reproduce the issue so we can help diagnose the problem.

### If you are having trouble understanding how to use the system

Please explain what you're trying to achieve.
Please provide clear and detailed instructions using resources like Gist so we can understand your situation.

### Questions related to customization

**Shiguredo members or administrators will not respond to customization-related questions**

- For customization inquiries, we will reply: "Shiguredo members or administrators do not handle customization-related questions."
- Feel free to ask questions on Discord, but please be aware that we won't provide customization support.

### Questions related to building/compilation

**Shiguredo members or administrators will not respond to build-related questions**

- Please use the pre-built binaries instead.
- If you wish to build the software yourself, please refer to the GitHub Actions configuration for guidance.
- For build-related questions, we will reply: "Shiguredo members or administrators do not handle build-related questions."
- Feel free to ask questions on Discord, but please be aware that we won't provide build support.
